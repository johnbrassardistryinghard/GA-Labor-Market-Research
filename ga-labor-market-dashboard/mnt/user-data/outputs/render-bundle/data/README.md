# Data Folder

Place `ststdsadata.xlsx` (BLS LAUS state data) in this folder before deploying.

**Where to get it:**
- https://www.bls.gov/lau/ → Tables → Statewide → "Statewide and Area Employment, Hours, and Earnings" → ststdsadata.xlsx

The dashboard expects exactly this filename. Update `BLS_FILE` in `ga_labor_dashboard_v2_render.py` if you rename it.
